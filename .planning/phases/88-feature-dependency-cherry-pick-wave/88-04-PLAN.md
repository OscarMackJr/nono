| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-88-15 | Denial of Service | PTY ctrl-z hang under supervised execution | mitigate | 4179ce03 fix: signal_pty_foreground_group() + handle_pty_suspension() route SIGTSTP correctly; this is the upstream fix for the hang |
| T-88-16 | Elevation of Privilege | nix:: PTY signal functions compiled on Windows (cfg gap) | mitigate | CRITICAL: ensure signal_pty_foreground_group() and handle_pty_suspension() are gated with #[cfg(any(target_os="linux", target_os="macos"))]; Windows compilation must not link nix:: |
| T-88-17 | Information Disclosure | detect_ci_provider() env-var enumeration | accept | Reads standard CI env vars (CI, GITHUB_ACTIONS, GITLAB_CI etc.); returns a static string identifying the provider; no secrets accessed |
