## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user `--workspace` input → child CWD + writable grant | Untrusted path crosses into the enforcement boundary; must be canonicalized + component-compared, never string-matched |
| resolved interpreter set → coverage gate | A covered wrapper that spawns an uncovered interpreter = partial confinement (ENG-02 core threat) |
| workspace NTFS owner → mandatory-label relabel | Non-relabelable workspace → opaque confined-write failure (R-B3) unless gated pre-launch |
