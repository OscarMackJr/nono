<objective>
Add the `nono override request` CLI affordance (CLI-01, Nono repo): a Rust-native command that surfaces the false-positive denial context (paths/domains, repo) from `DiagnosticFormatter` and emits a structured JSON request bundle for the out-of-nono approver/KMS-signing pipeline. </threat_model>
