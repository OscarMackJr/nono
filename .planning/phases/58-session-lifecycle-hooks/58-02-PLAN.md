## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| hook script → hook runtime | Hook script is an untrusted executable run outside the sandbox; its output (env file) crosses into trusted parent |
| env file → execute_sandboxed | Low-IL hook writes env file; Medium-IL parent reads and injects vars into child session |
| script path → validation | User-supplied hook path from profile; must be validated before exec |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-58-02-01 | Elevation of Privilege | env-file export — is_dangerous_env_var filter | mitigate | read_env_file applies crate::exec_strategy::is_dangerous_env_var() to every exported key; PATH, LD_PRELOAD, DYLD_*, NODE_OPTIONS etc. are all blocked at the filter |
| T-58-02-02 | Tampering | hook script in world-writable directory | mitigate | validate_hook_script checks parent directory is not world-writable (`parent.metadata()?.permissions().mode() & 0o002 == 0` on Unix); returns Err if world-writable |
| T-58-02-03 | Spoofing | hook script not owned by current user | mitigate | validate_hook_script checks uid == geteuid(); returns Err if mismatch — prevents shared-writable hook substitution |
| T-58-02-04 | Tampering | symlink swap between validate and spawn (TOCTOU) | accept | canonicalize() resolves symlinks at validation time; residual TOCTOU risk is the same as upstream's accepted trade-off (documented in ADR); full fd-relative hardening is REQ-UNDO-TOCTOU-01 scope |
| T-58-02-05 | Denial of Service | hanging before-hook blocks session start indefinitely | mitigate | run_hook with recv_timeout on timeout_secs; if timed-out → kill_process_group (SIGTERM + SIGKILL) → return Err (fail-closed per D-03) |
| T-58-02-06 | Repudiation | after-hook failure silently swallowed | mitigate | D-04: after-hook Err propagated via ? from execute_after_hook; caller in execution_runtime.rs surfaces it as error! log + non-zero exit |
| T-58-02-07 | Tampering | EnvFileGuard CREATE_NEW race | mitigate | create_new(true) on OpenOptions maps to O_EXCL on Unix; fails if file already exists; prevents pre-created env file injection |
| T-58-02-08 | Information Disclosure | hook env-var values persist in-memory after child spawn | mitigate | zeroize::Zeroize::zeroize called on each hook env-var value after injection into env_vars; limits the in-memory lifetime of credential values returned by execute_before_hook |
| T-58-02-SC | Tampering | npm/pip/cargo installs | accept | No new external packages in this plan; nix crate is an existing workspace dependency |
</threat_model>
