## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| kernel pre-create callback → ring buffer → worker thread | the only path the instrumentation touches; untrusted file-open requests cross into the kernel callback |
| kernel ↔ user-mode policy client (`\NonoPolicyPort`) | ABI-locked `NONO_IPC_REQUEST` wire struct crosses here; timing state must NOT |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-63-01 | DoS | NonoPreCreate / NonoWorkerThread | mitigate | No `Zw*`/`Nt*` file APIs added; `KeQueryPerformanceCounter` is HAL-only (recursive-I/O BSOD avoided) |
| T-63-02 | DoS | FltSendMessage span | mitigate | `timeout.QuadPart = -5000000LL` (500ms finite timeout) preserved; QPC brackets the call, does not change it |
| T-63-03 | DoS | callback + worker IRQL | mitigate | `KeQueryPerformanceCounter` is any-IRQL [CITED]; no new allocation/lock; valid at ≤APC_LEVEL and PASSIVE_LEVEL |
| T-65-ABI | Tampering | NONO_IPC_REQUEST wire struct | mitigate | Timing rides kernel-internal `NONO_RING_ENTRY.EnqueueQpc` only; `C_ASSERT(sizeof==532)` + Rust mirror untouched (defect-#13 avoidance) |
| T-65-PERTURB | Information disclosure (false measurement) | QPC-bracketed spans | mitigate | No `DbgPrint`/logging inside the timed span; accumulate raw ticks, sort+dump once at unload (integer-µs math, no kernel FP) |
| T-63-05 | Tampering (test-signed binary leakage) | VM-local `.sys` | mitigate | `.sys`/`.cat` stay VM-local, never committed; `git ls-files` gate confirms empty |
| T-65-SC | Tampering | npm/pip/cargo installs | accept | No package installs in this plan (kernel C + Azure CLI only; both pre-present) |
</threat_model>
